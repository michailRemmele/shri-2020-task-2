# Яндекс ШРИ. Задание 2. Реализация линтера

### Используемые зависимости
1. [json-to-ast](https://www.npmjs.com/package/json-to-ast) — Используется для построения AST по JSON-у.
Это позволяет получить подробную информацию о JSON-е подаваемом на вход линтера,
включая информацию о позиции (строка, столбец) блоков, элементов и прочих значений внутри анализируемого JSON-а.
