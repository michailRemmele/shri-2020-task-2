# Яндекс ШРИ. Задание 2. Реализация линтера

### Подготовка к работе
1. Установите Node (используется версия [v10.18.0](https://nodejs.org/en/blog/release/v10.18.0/))
2. Установите зависимости проекта:

    ```sh
    $ npm install
    ```

### Как собрать

Выполните следующую команду:

```sh
$ npm run build
```

### Как тестировать

Выполните следующую команду:

```sh
$ npm run test
```

### Используемые зависимости
1. [json-to-ast](https://www.npmjs.com/package/json-to-ast) — Используется для построения AST по JSON-у.
Это позволяет получить подробную информацию о JSON-е подаваемом на вход линтера,
включая информацию о позиции (строка, столбец) блоков, элементов и прочих значений внутри анализируемого JSON-а.
2. [@types/json-to-ast](https://www.npmjs.com/package/@types/json-to-ast) — Содержит описание интерфейсов и типов,
необходимых для использования пакета **json-to-ast** в TypeScript.

Остальные зависимости используются только для разработки:
поддержка языка TypeScript, сборка проекта, написание юнит тестов, линтинг кода, прекоммит хуки.
